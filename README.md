# react-native-dsphoto-module

DSphoto Wrapper

## Installation

```sh
npm install react-native-dsphoto-module
```

## Usage

```js
import DsphotoModule from "react-native-dsphoto-module";

// ...

const result = await DsphotoModule.multiply(3, 7);
```
